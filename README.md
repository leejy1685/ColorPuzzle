Flood Fill 알고리즘의 컬러 퍼즐 게임입니다.

## 🤔  프로젝트 소개

 **게임 특징**

- **스테이지 제작 기능** : Editor를 통해서 스테이지를 제작할 수 있습니다. 일종에 스테이지 개발 툴과 같은 역할을 합니다.
<img width="809" height="456" alt="image" src="https://github.com/user-attachments/assets/e27dd0b2-a7d7-4295-98b6-5ac1704fbee7" />

빌드 시에는 스테이지 개발 버튼을 비활성화 하고 빌드합니다.


- **컬러 퍼즐 기능** : 클릭 또는 터치한 칸과 인접한 같은 색상은 여러분이 선택한 색으로 변합니다. 이 기능을 이용해서 모든 칸을 타겟 컬러로 바꿔보세요.
